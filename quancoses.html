HTML

HTML stands for HyperText Markup Language and is used to build the structure of a document.

Elements

Elements are used in HTML to help separate, distinguish, categorize, etc. parts of a webpage.
Syntax for an element: < element type > stuff < / element type >

Examples of Elements

Paragraph Element - This is a block level element, so it will space itselp with other elements.
<p> your paragraph... </p>
Division Element - Much like a paragraph element, this is a block-level element allowing for division of sections.
<div> your content... </div>
h1 through h6 Elements - These are header elements. There can only be a single h1 on a page, and the numerical value must be in ascending order. e.g. h2 must come before an h3, and an h3 must come before an h4.
<h1> your main header </h1>
<h2> your subheader </h2>

HTML Boilerplate

This is standardized markup that needs to be in every html document.

Boilerplate Parts

Doctype - This tells the broswer what type of document it is reading.
<!DOCTYPE html>
HTML Element - This is the top-level element. Everything needs to go in here.
<html lang="en"> all your shit </html>
Title - This is the title of the page which is displayed in the browser tab. Browsers use the title element to create the links in the search results.
<title> your title </title>
Head - This is where all the metadata wil go.
<head> metadata </head>
Body - This is where all the content of the document will go.
<body> touch my body </body>
